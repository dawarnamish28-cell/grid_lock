# GRID_LOCK
Fast-paced grid-based shooter built in pure HTML.

▶ Play now: https://grid-lock-black.vercel.app

Basic Controls:
. WASD - Move 
· Mouse - Aim 
· LMB - Fire 
· RMB - ADS
. R - Reload 
· Q - Cycle 
· G - Grenade 
· F - Pickup 
· Space - Jump 
. Tab - Stats

Features:
- Zero frameworks
- Single-file HTML
- Tactical grid combat
